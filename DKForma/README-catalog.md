# Каталог FORMA DK

Готовый макет находится в `catalog.html`.

Как сохранить PDF:

1. Открой `catalog.html` в браузере.
2. Нажми кнопку `Сохранить в PDF`.
3. В окне печати выбери `Save as PDF`.
4. Формат: `A4`, ориентация: `Landscape / Альбомная`.
5. Поля: `None / Без полей`, фоновые изображения включить.

Привязка фото:

- `share-photo-27974710490.jpeg` - полнопиленая.
- `share-photo-27433199578.jpeg` - колото-пиленая.
- `share-photo-27911182298.jpeg` - галтованная.
- `share-photo-27974820826.jpeg` - вибролоток «Аура».

Позиция `100Х300Х30 колото-пиленая` оставлена как в исходном Word-прайсе. Если это опечатка и нужно `100х500х30`, поправить можно в `catalog-data.js`.
